# 📱 Mobile App – 780580

Yerelde Uygulama Geliştirme Challenge kapsamında React Native kullanılarak geliştirilmiş mobil uygulamadır.  
Bu proje, hasta değerlendirme ve tele-diş hekimliği süreçlerini mobil ortama taşıyarak dijital, erişilebilir ve sistematik bir yapı sunmayı amaçlamaktadır.

## 🚀 Proje Özeti

Uygulama temel olarak:

- 📸 Ağız içi görsellerin görüntülenmesi  
- 🦷 Dental değerlendirme ekranları  
- 📝 Genel değerlendirme formları  
- 👩‍⚕️ Tele-Diş Hekimliği giriş sistemi  
- 👥 Hasta listesi yönetimi  
- 👤 Kullanıcı profil ekranı  

modüllerini içermektedir.

Manuel ilerleyen hasta değerlendirme süreçlerinin mobil ortamda kayıt altına alınmasını ve yönetilmesini sağlar.

---

## 🛠️ Kullanılan Teknolojiler

- React Native  
- Expo (ön geliştirme süreci)  
- Android Studio (Build & APK üretimi)  
- React Navigation  
- Context API  
- AsyncStorage
